# 📘 Ejercicios de Sistemas Informáticos

Este repositorio contiene una colección de ejercicios prácticos de **Sistemas Informáticos**, organizados por temas y diseñados para reforzar conceptos clave de la asignatura.  

---

## 📂 Contenido

- **🔧 Administración de sistemas**
  - Comandos básicos de Linux/Windows
  - Gestión de procesos y servicios
  - Usuarios, permisos y seguridad

- **🌐 Redes**
  - Configuración de direcciones IP
  - Protocolos de comunicación
  - Herramientas de diagnóstico (ping, traceroute, netstat)

- **💾 Almacenamiento**
  - Sistemas de archivos
  - Montaje y gestión de discos
  - Copias de seguridad

- **⚙️ Virtualización**
  - Máquinas virtuales
  - Contenedores
  - Conceptos de cloud computing

---

##  Objetivos

- Reforzar los conocimientos teóricos mediante la práctica.  
- Familiarizarse con herramientas y comandos de administración de sistemas.  
- Desarrollar habilidades de resolución de problemas en entornos reales.  

---

##  Requisitos

- Sistema operativo: Linux (Ubuntu recomendado) o Windows.  
- Herramientas básicas:  
  - Git  
  - Editor de texto (VS Code, Vim, Nano)  
  - Acceso a terminal o consola
